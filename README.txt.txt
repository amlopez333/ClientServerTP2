Instalacion
1. En windows descargar del siguiente link -> https://www.python.org/downloads/release/python-352/
2. En linux debian based ->  sudo apt-get install python3.4

Para correr el programa
1. Abrir dos terminales o lineas de comando.
2. Cambiar al directorio del proyecto en ambas.
3. En una, correr python Servidor.py
4. En otra, correr python Cliente.py
5. Si da error, es porque python no esta en el path. Asi que agregarlo.
	ver -> http://stackoverflow.com/questions/3701646/how-to-add-to-the-pythonpath-in-windows-7 para windows
	ver -> http://stackoverflow.com/questions/3402168/permanently-add-a-directory-to-pythonpath para linux
6. Digitar nombre de usuario y dejar que el servidor lea su fortuna!
